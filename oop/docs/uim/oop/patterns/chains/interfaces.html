    Module uim.oop.patterns.chains.interfaces

    Interfaces

    NameDescription
    IAsyncHandler Async handler interface for asynchronous processing.
    IChainBuilder Chain builder interface for constructing handler chains.
    IChainCoordinator Chain coordinator interface for managing multiple chains.
    IConditionalHandler Conditional handler interface that can filter requests.
    IFallbackHandler Fallback handler interface for default processing.
    IGenericHandler Generic handler interface with typed request and response.
    IHandler Handler interface for the Chain of Responsibility pattern. Each handler decides whether to process the request or pass it along the chain.
    ILoggingHandler Logging handler interface for tracking request flow.
    IMiddleware Middleware handler interface with before/after hooks.
    IPriorityHandler Priority-based handler interface.
    IRequest Request interface for chain processing.
    IResponse Response interface for chain processing results.
