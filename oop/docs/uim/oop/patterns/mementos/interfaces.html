    Module uim.oop.patterns.mementos.interfaces

    Interfaces

    NameDescription
    ICaretaker Caretaker interface that maintains a collection of mementos. The caretaker never examines or modifies the contents of a memento.
    ICompressedMemento Compressed memento interface for memory-efficient storage.
    IGenericMemento Generic memento interface with typed state.
    IGenericOriginator Generic originator interface with typed state.
    IHistoryManager History manager interface for managing undo/redo with mementos.
    IMemento Memento interface that stores the internal state of an object. The memento is opaque to other objects except the originator.
    IMultiLevelUndoManager Multi-level undo manager interface.
    IOriginator Originator interface for objects that can create and restore mementos. The originator creates a memento containing a snapshot of its current state and uses it to restore its state later.
    ISnapshotManager Snapshot manager interface for creating named snapshots.
    IVersionedMemento Versioned memento interface with version tracking.
